## Installation

### Prérequis
- Node.js version 20.x ou supérieure
- pnpm version 8.x ou supérieure

### Commandes
```bash
# Installer les dépendances
pnpm install

# Lancer le script legacy
pnpm legacy

# Lancer les tests
pnpm test

# Linter le code
pnpm lint

# Formater le code
pnpm format
```
