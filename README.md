## Installation

### Requirements
- Node.js version 20.x or higher
- pnpm version 8.x or higher

### Commands
```bash
# Install dependencies
pnpm install

# Run the legacy script
pnpm legacy

# Run the tests
pnpm test

# Lint the code
pnpm lint

# Format the code
pnpm format
```
